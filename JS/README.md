# JavaScript Basic Practice

This folder contains small beginner-friendly examples for:

- callbacks
- promises
- async / await
- setTimeout
- setInterval
- ES modules

## How to Practice

1. Open `index.html` in your browser.
2. Right-click the page and choose **Inspect**.
3. Open the **Console** tab.
4. Click the buttons on the page and watch the output.

You can also open each `.js` file and read the comments.

## Files

- `callbacks.js` - passing one function into another function
- `promises.js` - basic promise resolve and reject examples
- `async-await.js` - using async functions with await
- `timers.js` - examples of `setTimeout` and `setInterval`
- `modules/mathUtils.js` - exported helper functions
- `modules/app.js` - importing and using module functions
- `index.html` - simple browser practice page
